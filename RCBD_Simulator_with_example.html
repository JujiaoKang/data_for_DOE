试验设计模拟：CRD vs RCBD

模型：观测产量 (yij) = 品种真实能力 (μ+τi) + 区组环境干扰 (βj) + 纯随机误差 (εij)

预设配置
1. 真实产量 (μ + τi)
2. 干扰 (βj) 与误差 (εij)
3. 选择方案并运行
CRD 完全随机
RCBD 区组设计

等待试验运行...

— 虚线 代表设定真实产量。误差棒 (±1s) 为计算所得波动。

核心计算公式

1. 单株观测值生成模型 (yij)
yij = (μ + τi) + βj + εij
2. 品种均值估计 (Sample Mean)
ȳi. = ( ∑ yij ) / n
即图表中“彩色柱子的高度”。将该品种所有的观测值相加并取平均。
3. 样本方差 (Sample Variance)
s² = ∑ ( yij - ȳ )² / ( n - 1 )
即图表中“白色误差棒的长度(s)”。
▶ 在 CRD 中： 肥力差异 βj 没被剥离，直接转化为极大的方差 s²。
▶ 在 RCBD 中： 方差分析(ANOVA) 剔除了已知区组效应后，均方误差(MSE) 仅剩纯随机误差，误差棒缩短。
