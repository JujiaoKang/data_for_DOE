核心计算公式与解析

1. 单株观测值生成模型 (yij)
yij = (μ + τi) + βj + εij
2. 品种均值估计 (Sample Mean)
ȳi. = ( ∑ yij ) / n
3. 样本方差 (Sample Variance)
s² = ∑ ( yij - ȳ )² / ( n - 1 )
即图表中“白色误差棒的长度(s)”。
▶ 错误设计： 均值严重偏移。因同品种环境完全一致，s²很小，制造了“高精度”假象。
▶ CRD： 肥力差异 βj 没被剥离，被动转化为极大的方差 s²。
▶ RCBD： 剔除了已知区组效应后，均方误差(MSE) 仅剩纯随机误差，精准且误差棒缩短。
